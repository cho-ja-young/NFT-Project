# NFT-Project

Truffle 과 Ganache 활용 NFT 블록체인 Project 입니다.
NFT Minting Page 를 만들어 이미지 업로드 및 NFT 등록을 진행합니다. 

## START

현재 로컬에서만 개발 중 ~ react 사용
```
npm start
```
